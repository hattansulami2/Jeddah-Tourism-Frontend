# Jeddah Tourism Website FrontEnd

A modern, responsive multi-page tourism portal dedicated to exploring the city of Jeddah. This project showcases the "Bride of the Red Sea" through a clean user interface, focusing on high-quality layouts and seamless navigation.

## 📌 Project Overview
The goal of this project was to build a visually engaging frontend that captures both the historical heritage and modern growth of Jeddah. It serves as a digital guide for tourists, featuring dedicated sections for history, culture, and major landmarks.

## 🚀 Key Features
*   **Responsive Multi-Page Design:** Fully functional pages for Home, History, and Attractions.
*   **Modern CSS Layouts:** Extensive use of **Flexbox** for navigation and **CSS Grid** for content galleries.
*   **Interactive Navigation:** A custom-built, mobile-friendly navigation bar.
*   **Semantic HTML:** Optimized for accessibility and clear structure.
*   **Professional Visuals:** Integration of icon libraries and optimized imagery to enhance UX.

## 🛠️ Tech Stack
*   **HTML5:** Structured semantic markup.
*   **CSS3:** Custom styling using Flexbox and Grid.
*   **Icons:** High-quality vector iconography.
*   **Editor:** Developed using VS Code.
